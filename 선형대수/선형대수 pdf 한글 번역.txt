생물학, 비즈니스, 화학, 컴퓨터 과학, 경제학, 전자, 공학, 물리학 및 사회 과학과 같은 많은 연구 분야의 실제 문제는 종종 선형 방정식 시스템을 푸는 것으로 축소될 수 있습니다.
선형 대수학은 이러한 시스템을 풀기 위한 체계적인 방법을 찾으려는 시도에서 비롯되었으며,
따라서 이 책은 선형 방정식을 공부하는 것으로 시작하는 것이 당연합니다.

a, b, c가 실수인 경우, 다음 형식의 방정식의 그래프
ax + by = c
는 직선이므로(a와 b가 둘다 0이 아닌 경우) 이러한 방정식을 변수 x와 y에서 선형 방정식이라고 합니다.
그러나 변수를 x1, x2, ..., xn으로 작성하는 것이 편리한 경우가 많습니다.
특히 두개 이상의 변수가 관련된 경우에는 더욱 그렇습니다.
a1x1 + a2x2 + ... + anxn = b 형식의 방정식을 n개의 변수 x1, x2, ..., xn에 대한 선형 방정식이라고 합니다. 여기서 a1, a2, ...는 실수(각각 x1, x2,...,xn의 계수라고 함)를 나타내고 b도 숫자(방정식의 상수항이라고 함)입니다.

변수 x1, x2, ..., xn에 대한 유한한 선형 방정식 모음을 이러한 변수에 대한 선형 방정식 시스템이라고 합니다.

그러므로
2x1 - 3x2 + 5x3 = 7은 선형 방정식입니다. x1, x2, x3의 계수는 각각 2, -3, 5이고, 방정식의 상수항은 7입니다.
선형 방정식의 각 변수는 첫번째 거듭제곱에만 발생합니다.->의역 선형 방정식의 각 변수는 1차항만 존재합니다, 차수가 1을 넘지 않습니다.

선형방정식 a1x1 + a2x2 + ... + anxn = b가 주어지면, n개의 수열 s1, s2, ..., sn은 a1s1+a2s2+... +ansn = b인 경우 방정식의 해라고 합니다.
즉, x1 = s1, x2 = s2, ... , xn = sn 대입 시 방정식이 충족되면.
시스템 안의 모든 방정식의 해라면, 수열은 연립 방정식의 해라고 합니다.
예를 들어 x = -2, y = 5, z = 0 와 x = 0, y = 4, z = -1 은
x + y + z = 3
2x + y + 3z = 1 모두 해다. 
시스템에는 해이 전혀 없거나 고유한 해이 있거나 무한한 해가 있을 수 있습니다. 예를 들어, 시스템 x + y = 2, x + y = 3은 두 숫자의 합이 동시에 2와 3이 될 수 없기 때문에 해가 없습니다.
해이 없는 시스템을 inconsistent 시스템이라고 합니다. 하나 이상의 해가 있는 시스템을 consistent이라고 합니다. 다음의 시스템
예제에는에는 무한히 많은 해가 있습니다.
